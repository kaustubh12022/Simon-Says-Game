# Simon Says Game

A modern, responsive web application of the classic memory game "Simon Says".
Built with HTML, CSS, and JavaScript.

## Features

-   **Classic Circular Design**: Authentic layout with 4 colored quadrants (Green, Red, Yellow, Blue).
-   **Visual Effects**: 3D shadows, neon glow animations, and tactile click feedback.
-   **Game Logic**: 
    -   Infinite levels with increasing difficulty.
    -   Strict user input validation.
    -   "Game Over" state with score display.
-   **Responsive**: Centers perfectly on different screen sizes.

## How to Play

1.  Open `index.html` in your web browser.
2.  Press any key to start the game.
3.  Watch the sequence of flashing colors.
4.  Repeat the sequence by clicking the correct buttons.
5.  The sequence gets longer with each level!

## Technologies Used

-   **HTML5**
-   **CSS3** (Flexbox, CSS Variables, Animations, Box-Shadows)
-   **JavaScript** (DOM Manipulation, Event Handling, Game Logic)

## Setup

No installation required. Simply clone the repository and open `index.html` to play.
